# Security policy

mailverdict is a stateless API: it never logs or stores checked email
addresses, and the entire dataset is public information aggregated from the
sources listed in the README.

## Reporting a vulnerability

Please report vulnerabilities privately via GitHub's security advisories
("Report a vulnerability" on the repository's Security tab). Include
reproduction steps; you'll get a response within a few days.

Please do not open public issues for security reports.
